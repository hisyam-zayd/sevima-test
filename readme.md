# Tes SEVIMA Fullstack Developer

Repo ini mencakup jawaban dan pengerjaan untuk bagian berikut:
1. *Proff of Concept* dari soal bagian 1 nomor 1 terkait memproses dan menyimpan integer sebesar 2048 bit
2. Bagian 2 nomor 2 terkait pembuatan aplikasi InstaApp (mirip instagram)

## Soal

### Bagian 1 nomor 1

Maksimum integer yang bisa disimpan dalam database dan diproses OS adalah 64 bit (BIGINT). Sevima membutuhkan cara untuk menyimpan dan memproses 2048 bit integer secara efisien. Berikan opsi metode yang bisa dipakai secara paling efisien dengan requirement sebagai berikut:
    - Bisa disimpan di database secara native
    - Bisa diproses di PHP dan bahasa lain secara native
    - 2048 bit bisa jadi 4096 bit jika diperlukan
    - Sertakan proof of concept metode penyimpanan dan metode perhitungan aritmetika (+ - x /)

### Bagian 2 nomor 2

Buatlah aplikasi InstaApp (mirip dengan Instagram) dengan, Scope:
1. Register dan Login
2. Posting text gambar
3. Like dan komentar
4. Autentifikasi pengguna
5. Hak akses terhadap post, like, dan komentar

Spesifikasi:
1. Bahasa/framework yang digunakan adalah PHP
2. Database dapat menggunakan RDBMS atau NoSQL
3. Menggunakan Frontend dan Backend (API) merupakan nilai tambah
4. UI/UX menarik merupakan nilai tambah

Pengumpulan:
1. Source code disimpan di github, kemudian sertakan linknya. (history git juga akan di nilai)
2. Buat video demo aplikasi (jelaskan aplikasi yang telah dibuat)
